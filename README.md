## ⚙️ Cómo trabajo el backend

- Diseño y desarrollo de APIs RESTful con ASP.NET Core.  
- Uso de **Entity Framework Core**, migraciones y LINQ para acceso a datos.  
- Implementación de autenticación/autorización (JWT, Identity) y validación de modelos.  
- Pruebas unitarias y de integración para asegurar calidad del código.  
- Contenerización con **Docker** y configuración de entornos de desarrollo y producción.  
- Flujo de trabajo con Git (GitFlow, ramas por feature, code reviews y documentación clara).
