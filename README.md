## ⚙️ Stack backend & forma de trabajar

- Diseño y desarrollo de APIs RESTful con ASP.NET Core.  
- Uso de **Entity Framework Core**, migraciones, LINQ y patrones Repository / Unit of Work.  
- Implementación de autenticación/autorización (JWT, Identity) y validación robusta de modelos.
